# Career Motivations Reflection & Growth Direction Assessment App
> Module arc generated by Claude in June 2026 based on direction from Jared and exploration / research into the core resources listed below.

A self-directed career-coaching curriculum bakes into a Metalab Sites app to answer "what actually drives me, and where am I going?"
Structured as the standard multi-module arc used by professional coaches (ICF-aligned, with senior-IC / engineering-leader inflection),
adapted as worksheets to be filled in at the driving individual's pace.

> **Not a march, a menu.** Doesn't have to be completed sequentially, but some modules are dependent on others
(6 depends on 5; and 5 depends on some lightweight 360 peer feedback)

## Sources synthesized

The arc is a synthesis of various frameworks and methodologies, not a single program:

- **ICF Core Competencies** — coaching-process backbone - https://coachingfederation.org/credentialing/coaching-competencies/icf-core-competencies/
- **Korn Ferry / BetterUp** — executive-coaching assessment-to-action structure - https://www.betterup.com/solutions/executive-coaching
- **Marshall Goldsmith** — stakeholder-centered coaching + feedforward + mini-360s - https://www.marshallgoldsmith.com/blog/categories/coaching
- **Stanford d.school — Designing Your Life** (Burnett & Evans) — workview/lifeview, Odyssey Plans, prototype conversations - https://lifedesignlab.stanford.edu/dyl
- **Schein's Career Anchors** — deep motivators - https://www.mpls.ox.ac.uk/files/training/careeranchors.pdf
- **CliftonStrengths (Gallup) + VIA Character Strengths** — talents and values lenses - https://www.gallup.com/cliftonstrengths/en/253715/34-cliftonstrengths-themes.aspx & https://www.viacharacter.org/character-strengths
- **Tanya Reilly — The Staff Engineer's Path** — senior-IC archetypes (tech lead / architect / solver / right hand) - https://www.oreilly.com/library/view/the-staff-engineers/9781098118723/
- **Lara Hogan** — manager-Voltron, sphere of influence, sustainment - https://larahogan.me/blog/manager-voltron/

## How to use each module

Every module file has the same shape:

1. **Purpose** — the question this module answers
2. **Essential references** — what to take/read first (free vs. paid noted)
3. **Exercises** — concrete things to do
4. **Worksheet** — fill-in section; my answers live here


## Pacing
Your own pace. Expect a solid 1/2-full day's effort for several of the modules; so the whole course is likely to take a couple of weeks to work through if you're doing it off the side of your desk / out of work hours.
Set deadlines for yourself that work for you.
(Future feature: scheduled reminders)

## Technical Setup
- React via React-router 
  - 🚀 Server-side rendering
  - ⚡️ Hot Module Replacement (HMR)
  - 📦 Asset bundling and optimization
  - 🔄 Data loading and mutations
  - 🔒 TypeScript by default
  - 🎉 TailwindCSS for styling
  - 📖 [React Router docs](https://reactrouter.com/)

- Metalab Sites (2.0) https://sites.metalab.com/docs
  - Branch-driven auto deploy w/ GitHub
  - User Identities via Okta Auth
  - User-specific content stored in localStorage only, because Sites db doens't have user-specific read security, and the content of the worksheets is likely to be personal information.
